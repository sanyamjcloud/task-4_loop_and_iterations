# Loops and Iterations Tasks

This repository contains Python tasks demonstrating loops and iterations.

## Files and Tasks

1. **task2_for_loop.py** – Print numbers 1–100 using a `for` loop.
2. **task3_while_countdown.py** – Countdown timer using a `while` loop.
3. **task4_break_continue.py** – Demonstrates `break` and `continue` statements.
4. **task5_iterate_string.py** – Iterate over characters in a string.
5. **task6_multiplication_table.py** – Generate a multiplication table for a number.
6. **task7_range_with_steps.py** – Use `range()` with step values.
7. **task8_loop_with_conditions.py** – Combine loops with conditional statements.
8. **task9_real_world_example.py** – Example: Shopping cart simulation using loops.
